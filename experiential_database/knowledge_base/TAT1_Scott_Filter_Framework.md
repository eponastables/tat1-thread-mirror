# TAT1 Knowledge Base: The Scott Filter Framework

**Source:** RC Research Report, ISSUE-014 (2026-03-20)

---

## Core Principle

The Scott Filter is a set of rules for eliminating horses that are unlikely to win, before any deep handicapping analysis begins. It is a tool for focusing attention on the genuine contenders.

## The Modernised Filters

| Filter | Rule | Modern Adaptation |
| :--- | :--- | :--- |
| **Class** | Eliminate horses with a poor class rating based on weight carried. | Supplement with Benchmark ratings and prize money. Adjust for interstate runners. |
| **Weight** | Penalise horses carrying excessive weight relative to their class. | Apply with caution in high-grade races where top weights are common. |
| **Form** | Eliminate horses with poor recent form (e.g., beaten by more than 5 lengths at last start). | Apply with context — consider excuses for poor runs (e.g., bad luck, unsuitable track). |
| **Wet Track** | Treat wet track form as a binary qualifier. | Assess on a spectrum (Soft 5-7 vs. Heavy 8-10). Some horses handle Soft but not Heavy. |
| **Field Size** | Avoid small fields. | Apply cautiously in high-grade races where small fields are common. |
| **Odds-On** | Never back a horse at odds-on. | Still widely respected. Some practitioners apply only to fields of 8+ runners. |

## Known Edge Cases

The filters are known to be less effective in:

*   Small provincial fields (4-6 runners)
*   Maiden races
*   Races with a high number of interstate runners

## The Research Gap

No peer-reviewed statistical test of the Scott Filters on post-2010 Australian racing data exists. This is a major opportunity for the project. The backtesting module being built by APT1 will allow us to produce original, evidence-based knowledge about the filters' modern efficacy.
