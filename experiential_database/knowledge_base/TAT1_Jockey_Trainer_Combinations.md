# TAT1 Knowledge Base: Jockey/Trainer Combinations

**Source:** RC Research Report, ISSUE-015 (2026-03-22)

---

## Core Principle

The value in jockey-trainer combinations is not in the elite partnerships (e.g., McDonald-Waller), which are fully priced by the market, but in the **secondary partnerships of positive-ROI trainers**.

## The Data Gap

No publicly available Australian source publishes ROI or POT data for specific jockey-trainer combinations. This is a key data gap that a StatFreaks subscription would fill.
